Here is how you can create words in Gujarati by joining characters.

  • ગ + ◌ુ + જ + ર + ◌ા + ત + ◌ી = ગુજરાતી
  • મ + સ + ◌્ + ત = મસ્ત
  • પ + ◌્ + ર + થ + મ = પ્રથમ
  • ક + ર + ◌્ + મ = કર્મ
  • ત + ◌્ + ણ = ત્રણ
